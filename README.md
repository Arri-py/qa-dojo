## Задание

### Шаг 1 — Поднять контейнер

Запусти калькулятор одной командой:

```bash
docker-compose up --build
```

Если всё прошло успешно — в терминале появится что-то вроде:

```
calculator  | INFO:     Application startup complete.
```

### Шаг 2 — Изучить API

Открой в браузере интерактивную документацию:

```
http://localhost:8000/docs
```

Перед тобой Swagger UI — документация с возможностью тестировать API прямо в браузере.  
Найди все эндпоинты калькулятора и попробуй каждый из них вручную.

### Шаг 3 — Написать консольный клиент

Напиши Python-скрипт `client.py`, который:

1. Показывает меню в консоли
2. Принимает от пользователя два числа и операцию
3. Отправляет запрос к API калькулятора
4. Выводит результат

Пример работы:

```
=== Калькулятор ===
1. Сложение
2. Вычитание
3. Умножение
4. Деление
0. Выход

Выберите операцию: 1
Введите первое число: 10
Введите второе число: 5
Результат: 15.0
```

> Для отправки HTTP-запросов используй библиотеку `requests`.  
> Установить: `pip install requests`
