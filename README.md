# QA Dojo

> *Додзё — место, где рождается мастерство.*

Здесь ты будешь учиться работать с Git, Docker и тестированием.  
Каждое задание — это **ката**: упражнение, которое нужно выполнить и понять.

Не гугли сразу. Сначала попробуй сам.

---

## Путь ученика

Каты идут по нарастающей — каждая строится на навыках предыдущей.  
Проходи по порядку.

| # | Название | Чему учит | Ветка |
|---|----------|-----------|-------|
| 01 | Калькулятор-сан | Docker Compose, REST API, HTTP-методы, версионирование API, venv | `kata/01-calculator` |

---

## Как приступить к кате

```bash
# Получить актуальное состояние репозитория
git fetch

# Переключиться на нужную ветку
git checkout kata/01-calculator
```

Открой `README.md` в ветке — там полное описание задания.

---

## Правила додзё

- Читай задание внимательно — подсказки спрятаны внутри
- Разбирайся как работает код, не только что он делает
- Если что-то не работает — сначала читай логи, потом гугли
