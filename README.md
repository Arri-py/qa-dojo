### Шаг 2 — Изучить API

Открой документацию в браузере:

```
http://localhost:8000/docs
```

Swagger UI покажет все эндпоинты — сгруппированные по версиям **v1** и **v2**.  
Попробуй каждый из них вручную прямо в браузере, прежде чем писать код.

#### Что умеет API

Калькулятор хранит **историю вычислений** — каждый результат получает уникальный `id`.

| Метод | Эндпоинт | Что делает |
|-------|----------|------------|
| `GET` | `/api/v1/health` | Проверить, жив ли сервис |
| `GET` | `/api/v1/operations` | Список доступных операций |
| `POST` | `/api/v1/calculations` | Создать вычисление |
| `GET` | `/api/v1/calculations` | История всех вычислений |
| `GET` | `/api/v1/calculations/{id}` | Получить вычисление по ID |
| `PUT` | `/api/v1/calculations/{id}` | Заменить вычисление (все поля) |
| `PATCH` | `/api/v1/calculations/{id}` | Изменить вычисление частично |
| `DELETE` | `/api/v1/calculations/{id}` | Удалить вычисление |

**v2** расширяет v1: добавляет операции `modulo` и `power`, в ответах появляется поле `expression` (`"10.0 × 5.0 = 50.0"`), и новый эндпоинт `GET /api/v2/calculations/{id}/expression`.

#### Чем v1 отличается от v2

| | v1 | v2 |
|---|----|----|
| Операции | `add subtract multiply divide` | + `modulo power` |
| Поле `expression` в ответе | нет | есть |

### Шаг 3 — Написать консольный клиент

Напиши Python-скрипт `client.py`, который работает с **v1** и:

1. Показывает меню с операциями
2. Принимает от пользователя два числа
3. Отправляет `POST /api/v1/calculations` и показывает результат
4. После вычисления предлагает:
   - посмотреть историю (`GET /calculations`)
   - изменить одно из чисел в последнем результате (`PATCH`)
   - удалить последний результат (`DELETE`)
   - выйти

Пример работы:

```
=== Калькулятор-сан ===
1. Сложение      (add)
2. Вычитание     (subtract)
3. Умножение     (multiply)
4. Деление       (divide)
0. Выход

Выберите операцию: 1
Введите первое число: 10
Введите второе число: 5

Результат: 15.0  (id: 3fa85f64-...)

Что дальше?
h — история  p — изменить число  d — удалить  q — выйти
> h

История вычислений (1):
  [3fa85f64-...] 10.0 add 5.0 = 15.0
```

> Для HTTP-запросов используй библиотеку `requests` (`pip install requests`).
